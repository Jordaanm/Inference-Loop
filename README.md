# Inference Loop

A terminal-based incremental game where you build and scale an AI commission business.

Start as a solo operator, manually completing simple jobs to earn your first dollars. Spend them on tokens to power cloud AI agents, automate your workflow, invest in hardware, and eventually run your own data center. Research new capabilities, unlock better models, and — when you're ready — research Time Travel to prestige and do it all again, faster.

Built with [Ink](https://github.com/vadimdemedes/ink) (React for CLIs) and TypeScript.

## Running the game

```bash
npm install
npm run dev
```

## Development

```bash
npm run build   # Compile TypeScript
npm test        # Run tests
npm run lint    # Lint with ESLint
```
